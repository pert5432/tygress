# Installation

Run `npm i tygress` or `yarn add tygress`

Make sure you have decorators enabled in your `tsconfig.json`
```json
{
  "compilerOptions": {
    "experimentalDecorators": true,
    "emitDecoratorMetadata": true
  }
}
```
